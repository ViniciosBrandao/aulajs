# aulajs

Repositório dedicado aos meus estudos em JavaScript, com exercícios práticos realizados durante aulas na UEG no curso de Sistemas para Internet e estudos individuais.

## 🚀 Objetivo

Consolidar fundamentos de programação e evoluir para desenvolvimento profissional, organizando todo o aprendizado de forma estruturada.

## 📂 Estrutura

- **fundamentos/** → conceitos básicos (variáveis, tipos, operadores)
- **controle/** → estruturas de decisão e repetição
- **funcoes/** → funções e escopo
- **arrays/** → manipulação de listas
- **objetos/** → estruturas de dados complexas
- **desafios/** → exercícios e problemas práticos

## 🛠️ Tecnologias

- JavaScript (Node.js)
- Git & GitHub
- WSL (Ubuntu)

## 📈 Progresso

Este repositório é atualizado continuamente conforme avanço nos estudos.
